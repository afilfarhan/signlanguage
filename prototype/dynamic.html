Architecture (PDR §7.1, S-06): webcam → MediaPipe Hands → 21×3 landmark sequence buffer (45 frames) → seq_features.normalize_sequence() ported to JS → Transformer (ONNX, 78,600 params, 360 KB) loaded with onnxruntime-web → softmax → top-k display + confidence-gated verdict. All inference is on-device; no frames or landmarks leave the browser.

Note: this preview's iframe has no network, so MediaPipe + ONNX-Runtime can't download. Open the file via localhost to use it for real. The model was trained on synthetic data — see ml/models/dynamic_robustness_report.md for the realistic robustness sweep.
